## 3. Observacoes por Suite
### 3.1 TestU01 (Rabbit/Alphabit/SmallCrush)
- Todos operaram sobre 1,048,576 bits gerados via streamer com whitening; os logs exibem `All tests were passed` e estatisticas dentro do regime esperado.
- Rabbit tomou 0.23 s de CPU (38 estatisticas); Alphabit completou praticamente instantaneo (17 estatisticas). SmallCrush consumiu ~5.2 s de CPU e validou 15 estatisticas.
- MPC-N recebeu os eventos `diagnostics.rabbit/alphabit/smallcrush` com status `completed`.

### 3.2 ENT
- Entropia ideal (7.999994). Chi-square 290.40 (superior a 6.31 % dos casos aleatorios), Monte Carlo pi erro 0.02 %, serial corr 2.4e-5.
- Resultado confirma ausencia de compressibilidade perceptivel no fluxo whitened.

### 3.3 rngtest
- 87 falhas em 100,000 execucoes FIPS (principalmente Long Run e Runs). A taxa esta pouco acima do limite tipico (max. esperado ~65 em 100k para 3 sigma).
- Script ja distingue `completed_with_failures`; precisamos investigar se o chunk de 8 KiB + whitening provoca padroes curtos repetidos ou se a duracao deve ser ampliada para amortizar ruido.

### 3.4 NIST STS
- Automacao alimentou apenas 10 streams (1,000,000 bits cada). Como `assess` continua configurado para sample size 1000, o relatorio acusa 0/1000 e p-value 0.000000 em todas as colunas.
- Para resultados validos precisamos (a) gerar 1000 streams (125 MB) ou (b) alterar a configuracao para refletir as 10 streams usadas. Ate la o relatorio serve apenas como evidencia de execucao, nao de aprovacao.

### 3.5 Testes internos
- `make test` passou com 103 testes e duas warnings de `scipy.stats` (skew/kurtosis com dados identicos). Nenhuma regressao funcional identificada.
- `make test-security` reforcou determinismo, avalanche 50.34 %, reversibilidade 100 % em 1 MB, sensibilidade a senha e consistencia multi-instancia.
- `make test-performance` confirmou throughput >=470 KB/s (dados reais) e >=550 KB/s em bloco de 5 MB, memoria adicional ~5.4 MB.
- `tests/validation/test_external_diagnostics.py` so coleta corretamente quando `PYTHONPATH=src`; registre o fluxo correto para evitar falsos negativos futuros.
