## QUICK START (5 MINUTOS)

### Instalação

```bash
# Clone o repositório
cd /home/kbe/KAYOS_SYSTEMS/KayosCrypto

# Teste o sistema
python3 kayoscrypto_evolved_final.py
```

### Uso Básico

```python
from kayoscrypto_evolved_final import KayosCryptoUltimate

# Criar instância
cipher = KayosCryptoUltimate(password="minha_senha_secreta")

# Encriptar
data = b"Dados importantes"
encrypted = cipher.encrypt(data)

# Decriptar
decrypted = cipher.decrypt(encrypted)

# Verificar
assert data == decrypted # Sempre funciona!
```

### CLI

```bash
# Encriptar arquivo
python3 kayoscrypto_cli.py encrypt arquivo.txt

# Decriptar arquivo
python3 kayoscrypto_cli.py decrypt arquivo.txt.kayos

# Benchmark
python3 kayoscrypto_cli.py benchmark
```
